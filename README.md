## Steps necessary to run the software

Go to Requirements.py

1. pip install -r requirements.txt
2. python run.py
3. On local host go to "http://127.0.0.1:5000" to see development server.

Copy of requirments.txt
    flask
    pymongo
    python-dotenv
    flask-login
    werkzeug
